# phan-tich-game-nhom-36
GAME ANALYTICS – VIDEO GAME SALES

1. Giới thiệu
Dự án này thực hiện phân tích dữ liệu doanh số game nhằm khám phá xu hướng thị trường, xác định các yếu tố ảnh hưởng đến doanh thu và rút ra các insight có giá trị.
2. Mục tiêu
- Phân tích doanh số theo năm
- Xác định thể loại game phổ biến
- Tìm game bán chạy nhất
- Phân tích theo nền tảng
- Trực quan hóa dữ liệu
3. Dataset
Dataset Video Game Sales bao gồm các thông tin: Name, Platform, Year, Genre, Publisher, Global_Sales.
4. Data Cleaning
- Xóa dữ liệu thiếu Year
- Chuyển Year sang int
- Xóa dữ liệu trùng
5. Phân tích dữ liệu (EDA)
- Doanh số tăng mạnh giai đoạn 2000–2010
- Thể loại Action phổ biến nhất
- Platform như PS2, Xbox chiếm ưu thế
6. Visualization
- Line chart: xu hướng theo năm
- Bar chart: thể loại và platform
7. Insight
- Thị trường game phát triển mạnh trong giai đoạn 2000–2010
- Game hành động chiếm ưu thế
- Console phổ biến hơn PC
8. Công cụ sử dụng
Python, Pandas, Matplotlib, Seaborn, Scikit-learn
9. Kết luận
Dự án giúp hiểu rõ xu hướng ngành game và có thể áp dụng trong kinh doanh.
